# Daily Meal Nutrition Calculator

A mobile-friendly web application for tracking daily nutrition by building meals and monitoring macronutrient intake.

## Features

- **Food Search**: Search for any food using the USDA FoodData Central API
- **Meal Builder**: Add multiple foods to create meals with customizable quantities
- **Daily Tracker**: Track multiple meals throughout the day (Breakfast, Lunch, Snacks, Dinner, or custom)
- **Nutrition Analysis**: View detailed macronutrients (calories, protein, carbs, fat, fiber, sugar) plus vitamins & minerals
- **Editable Daily Goals**: Customize your daily calorie and protein targets in the Daily Summary section
- **Smart Insights**: Get health hints based on your personalized daily intake goals
- **Local Storage**: All data persists in your browser - no account needed

## Setup Instructions

### 1. Prerequisites

Before deploying, ensure you have:

- **Node.js** (v18 or later) and **npm** or **pnpm** installed
- **dfx** (Internet Computer SDK) installed - [Installation Guide](https://internetcomputer.org/docs/current/developer-docs/setup/install/)
- **USDA FoodData Central API Key** - [Get your free API key](https://fdc.nal.usda.gov/api-key-signup.html)
